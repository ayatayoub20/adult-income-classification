# Adult Income Classification Project

## Project Overview

This project uses the Adult Income dataset to predict whether a person's income exceeds 50K per year.

## Objective

The goal is to build a classification model and analyze the factors that influence income levels.

## Dataset

Each row represents one individual and includes demographic and employment-related information such as age, education, occupation, marital status, and hours worked per week.

## Methods Used

- Data cleaning
- Exploratory data analysis
- Preprocessing
- SMOTE for class imbalance
- Random Forest Classification
- Permutation Importance

## Key Insights

- Higher education levels are associated with higher income.
- Individuals earning more than 50K tend to work more hours per week.
- Capital gain is one of the strongest predictors of income.

## Model Performance

The final Random Forest model achieved approximately 84% accuracy on the test set.
